# CSE534 Project1
---
## How to run

### Step by step operation

**Build docker image**

```bash
make build
```

**Create environment file**

```bash
make arch
```

**Deploy**

```bash
make deploy
```

**Show the net info**

```bash
make show_net
```

**Testing**

```bash
make test
make test1
```

**Down & Clean**

```bash
make down
make clean
```

### A handful of pokers

```bash
make all
```
